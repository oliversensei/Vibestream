# **App Name**: VibeStream

## Core Features:

- Interactive Sidebar: Dynamic sidebar navigation for easy access to playlists, settings, and other sections.
- Main Content Area: Displays a curated playlist with track details and interactive elements.
- Clickable Tracks: Tracks within the playlist are clickable, initiating playback.
- Bottom Player Bar: Dedicated player bar at the bottom for playback controls.
- Play/Pause Controls: Standard play/pause functionality with intuitive control buttons.
- Visualizer animation: Animation corresponding to the played song.

## Style Guidelines:

- Primary color: Saturated dark green (#1DB954) for a vibrant musical feel.
- Background color: Dark, desaturated green (#191414) to provide contrast.
- Accent color: A lighter green (#A6FFCB) as an accent, drawing the eye and balancing against the dark background.
- Font: 'Inter' (sans-serif) for both headlines and body text to provide a clean, modern interface.
- Use minimalist, custom-designed icons for the sidebar and player controls.
- Spotify-like layout with a sidebar, main content area, and bottom player bar.
- Smooth transitions and animations for interactive elements, such as the play/pause button and playlist scrolling.
